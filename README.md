# simple-spreadsheet-uploader

## Description:

Simple upload form that filters for spreadsheet (csv, xlsx) file types.

## Features:

- Simple upload submit
- File selection method
- File type filtering
- csv file format return
- Basic page routing

## Dependencies:

- express
- multer
- nodemon
- dotenv

## Build Instructions:

1. **Install dependencies**
```sh
npm install
```

2. **Quick Start:**
```sh
npm start
```

3. **Development start:**
```sh
npm run start:dev
```
